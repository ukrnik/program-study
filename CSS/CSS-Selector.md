
В CSS существует множество селекторов, которые позволяют выбирать элементы для применения стилей. Вот основные категории и их примеры:

## 1. Простые селекторы

- **Элемент (type selector)**: выбирает все элементы заданного типа.

```css
div {
  color: red;
}
```

- [[CSS-класс|Класс (class selector)]]: выбирает все элементы с указанным классом.

```css
.my-class {
  font-size: 16px;
}
```

- **ID (ID selector)**: выбирает элемент с указанным уникальным ID.

```css
#my-id {
  background-color: blue;
}
```

- **Универсальный (universal selector)**: выбирает все элементы.

```css
* {
  margin: 0;
}
```

## 2. Комбинированные селекторы

- **Селектор потомков**: выбирает элементы, которые являются потомками указанного родителя.

```css
div p {
  color: green; /* Все <p>, находящиеся внутри <div> */
}
```

- **Селектор дочерних элементов**: выбирает прямых потомков указанного элемента.

```css
div > p {
  font-weight: bold; /* Все <p>, являющиеся дочерними для <div> */
}
```

- **Соседний селектор**: выбирает элемент, который идет сразу после другого элемента.

```css
h1 + p {
  color: orange; /* Первое <p> сразу после <h1> */
}
```

- **Общий соседний селектор**: выбирает все элементы, расположенные после указанного.

```css
h1 ~ p {
  font-size: 14px; /* Все <p>, идущие после <h1> */
}
```

## 3. Селекторы атрибутов

- С элементом, имеющим атрибут:

```css
[type] {
  border: 1px solid black;
}
```

- С определенным значением атрибута:

```css
[type="text"] {
  background: yellow;
}
```

- С атрибутом, значение которого начинается с:

```css
[href^="https"] {
  color: blue;
}
```

- С атрибутом, значение которого заканчивается на:

```css
[href$=".pdf"] {
  color: red;
}
```

- С атрибутом, содержащим определенное значение:

```css
[class*="btn"] {
  padding: 10px;
}
```

## 4. Псевдоклассы

- **Динамические состояния**:

```css
a:hover {
  text-decoration: underline;
}
input:focus {
  outline: 2px solid green;
}
```

- **Структурные селекторы**:

```css
:first-child {
  color: red; /* Первый ребёнок */
}
:last-child {
  color: green; /* Последний ребёнок */
}
:nth-child(2) {
  color: blue; /* Второй ребёнок */
}
```

## 5. Псевдоэлементы

- **Для добавления элементов**:

```css
p::before {
  content: "★";
  color: gold;
}
p::after {
  content: "✦";
  color: silver;
}
```

- **Для работы с частями текста**:

```css
::first-letter {
  font-size: 2em;
}
::first-line {
  font-weight: bold;
}
```

## 6. Групповые селекторы

- **Объединяют несколько элементов в одну группу**:

```css
h1, h2, h3 {
  font-family: Arial, sans-serif;
}
```

## 7. Селекторы с псевдо-атрибутами

- **not для исключения**:

```css
div:not(.active) {
  opacity: 0.5;
}
```

- **Селекторы состояния**:

```css
input:checked {
  border: 2px solid green;
}
```
