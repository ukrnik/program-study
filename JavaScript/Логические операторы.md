## Описание

Логические операторы позволяют комбинировать или изменять логические значения (true/false). В JavaScript их три:

- && — И (AND)
- || — ИЛИ (OR)
-  ! — НЕ (NOT)
---
## 1. Оператор && (И)

Возвращает true, **если оба операнда истинны**. В противном случае — false.

```js
выражение1 && выражение2
```

| **Выражение**  | **Результат** |
| -------------- | ------------- |
| true && true   | true          |
| true && false  | false         |
| false && true  | false         |
| false && false | false         |

---
## 2. Оператор || (ИЛИ)

Возвращает true, **если хотя бы один из операндов истинен**. Возвращает false, если **оба операнда ложны**.

```js
выражение1 || выражение2
```

| **Выражение**    | **Результат** |
| ---------------- | ------------- |
| true \|\| true   | true          |
| true \|\| false  | true          |
| false \|\| true  | true          |
| false \|\| false | false         |

---
## 3. Оператор ! (НЕ)

> [!info]
> Инвертирует логическое значение:
> - !true → false
> - !false → true

---
## Комбинирование логических операторов

Можно комбинировать операторы &&, || и ! в одном выражении:

```js
const age = 25;
const isMember = true;

if ((age >= 18 && isMember) || age >= 65) {
  console.log("Скидка предоставлена");
} else {
  console.log("Скидки нет");
}
```

- Если age >= 18 && isMember — true **или** age >= 65 — true, выводится "Скидка предоставлена".
---
## Приоритет операторов:

Приоритет выполнения логических операторов:

1.  ! (НЕ)
2. && (И)
3. || (ИЛИ)

```js
console.log(!false && true || false);
// Выполняется так: (!false) && true || false
// Сначала !false → true
// Затем true && true → true
// Затем true || false → true
```

---
## Короткое замыкание (Short-circuiting)

Логические операторы могут завершить выполнение раньше:
- && возвращает **первое false значение** или последнее true.
- || возвращает **первое true значение** или последнее false.

```js
console.log(0 && "Hello");  // 0
console.log(1 && "Hello");  // "Hello"
```

```js
console.log(0 || "Hello");  // "Hello"
console.log(1 || "Hello");  // 1
```
