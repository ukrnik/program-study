# Методы Date (get) — получение даты и времени

Методы **`get`** позволяют получать отдельные части даты и времени из объекта `Date`.

> [!warning] Важно
> Месяцы считаются с **0** (январь) до **11** (декабрь).
> Дни недели считаются с **0** (воскресенье) до **6** (суббота).

---
## Таблица методов

| Метод                | Описание                                      | Пример результата |
| -------------------- | --------------------------------------------- | ----------------- |
| `.getFullYear()`     | Возвращает год                                | `2025`            |
| `.getMonth()`        | Возвращает месяц (0–11)                       | `7` (август)      |
| `.getDate()`         | Возвращает число месяца (1–31)                | `15`              |
| `.getDay()`          | Возвращает день недели (0–6, 0 — воскресенье) | `5` (пятница)     |
| `.getHours()`        | Возвращает часы (0–23)                        | `10`              |
| `.getMinutes()`      | Возвращает минуты (0–59)                      | `30`              |
| `.getSeconds()`      | Возвращает секунды (0–59)                     | `0`               |
| `.getMilliseconds()` | Возвращает миллисекунды (0–999)               | `0`               |
| `.getTime()`         | Возвращает метку времени (мс с 1970-01-01)    | `1755250200000`   |
| `Date.now()`         | Текущее время в миллисекундах                 | `1755250200000`   |

---
## Примеры использования

```js
const date = new Date(2025, 7, 15, 10, 30, 0);

console.log(date.getFullYear());     // 2025
console.log(date.getMonth());        // 7 (август, так как месяцы начинаются с 0)
console.log(date.getDate());         // 15
console.log(date.getDay());          // 5 (пятница)
console.log(date.getHours());        // 10
console.log(date.getMinutes());      // 30
console.log(date.getSeconds());      // 0
console.log(date.getMilliseconds()); // 0
console.log(date.getTime());         // 1755250200000

// Альтернатива без объекта Date
console.log(Date.now());             // 1755250200000

```
