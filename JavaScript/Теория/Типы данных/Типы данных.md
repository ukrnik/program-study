Типы данных используются для хранения и обработки различных видов информации в программе.

---
## **Оператор** `typeof`

Оператор `typeof` используется для определения типа данных значения или выражения.

```js
typeof operand
```


```js
const quantity = 17;
console.log(typeof quantity); // виведе "number"

const message = "JavaScript is awesome!";
console.log(typeof message); // виведе "string"

const isSidebarOpen = false;
console.log(typeof isSidebarOpen); // виведе "boolean"

let username;
console.log(typeof username); // виведе "undefined"

let status = null;
console.log(typeof status); // виведе "object"
```

---
## **Number**

Числа могут быть положительными или отрицательными, целыми или дробными. Целая и дробная части числа разделяются точкой.

```js
const age = 20;
const salary = 3710.84;
```

<center><font color="#d83931">Для дроби мы используем точку</font></center>

---
## **String**

Строка (String) является последовательностью символов, вложенных в одинарные ' ' или двойные кавычки "".

```js
const username = 'Mango995';
const description = "JavaScript is awesome!";
```

---
## **Boolean**

Логический тип данных (Boolean) имеет только два значения: true и false.

```js
const isModalOpen = true;
const isLoggedIn = false;
```

---
## Null

Является явным указанием на отсутствие значения. Это намеренно присваиваемое значение для обозначения пустого или несуществующего объекта.

```js
let value = null;
console.log(value); // null
```

---
## Undefined

Указывает, что переменная объявлена, но её значение не задано. Это значение по умолчанию для неинициализированных переменных или отсутствующих аргументов функции.

```js
let value;
console.log(value); // undefined
```

---

<center><font color="#d99694">Ключевое различие: undefined — это “неопределённость” (что-то ещё не определено), а null — это “ничего” (намеренное отсутствие).</font></center>


```js
let a;
console.log(a); // undefined

let b = null;
console.log(b); // null
```
