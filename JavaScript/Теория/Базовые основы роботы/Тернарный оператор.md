## Описание

Это **тернарный оператор** (? :) в JavaScript. Он является краткой формой для [[if...else]].

---
## Синтаксис

```js
условие ? выражение_если_истинно : выражение_если_ложно;
```

- условие — это логическое выражение (true или false).
- выражение_если_истинно — выполняется, если условие истинно (true).
- выражение_если_ложно — выполняется, если условие ложно (false).
---
## Пример №1

```js
const age = 16;

const access = age >= 18 ? "Доступ разрешён" : "Доступ запрещён";

console.log(access); // "Доступ запрещён"
```
---
## Пример №2

```js
const hour = 10;

const greeting = hour < 12 ? "Доброе утро" : "Добрый день";

console.log(greeting); // "Доброе утро"
```
---
## Пример №3

```js
const isLoggedIn = true;

const message = isLoggedIn ? "Добро пожаловать!" : "Пожалуйста, войдите";

console.log(message); // "Добро пожаловать!"
```
---
## Пример №4

```js
const a = 5;
const b = 10;

const min = a < b ? a : b;

console.log(min); // 5
```
