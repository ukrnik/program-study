# Есть два вида подключения скрипта:

## 1. Внешний

### Синтаксис

```html
<script src="my-script.js" defer></script>
```
### Атрибуты:

`scr` - указываем путь к файлу JavaScript (.js)

`defer` - с этим атрибутом можно вставлять скрипт и в тег `<head>` и в `<body>`. А также указывает на то что скрипт JavaScript должен быть загружен после загрузки html (это позволяет нам избежать задержки отображения страницы)

`type=“module”` - указывает, что скрипт является модулем ECMAScript (ES6+). 
	1. браузер загружает модульный скрипт параллельно с остальной страницей и выполняет его после построения [[DOM]].
	2. Код внутри модуля выполняется в собственном пространстве имен. Это значит, что переменные и функции, объявленные внутри модуля, не засоряют глобальную область видимости.
	3. Код в модулях всегда выполняется в строгом режиме JavaScript ([[Strict mode|strict mode]]), даже если это не указано явно.

```html
<!DOCTYPE html>
<html>
<head>
    <title>My HTML-page</title>
    <script src="my-script.js" defer></script> // ✅ норм
</head>
<body>
	<script src="my-script.js" defer></script> // ✅ норм
    <!-- Наполнение страницы -->
</body>
</html>
```

## 2. Встроенный

Синтаксис

```html
<script> код JS </script>
```

То есть мы можем выполнять кусочек кода прям в Html но только в внутри <head>
