## Конкатенация строк

Если применить оператор + к строке и любому другому типу данных, результатом будет новая строка, содержащая объединение исходных значений. Это называется конкатенация (склеивание).

```js
const message = "Mango " + "is" + " happy";
console.log(message); // "Mango is happy"
```

<center><font color="#ff0000">Строки "Mango " и " happy" содержат пробелы, чтобы текст сообщения был читабельным.</font></center>

Также мы можем использовать переменные для подставления данных.

```js
const age = 24;
const message = "Poly is " + age + " years old!";
```

При конкатенации любой тип данных будет приведен в строку и сопряжен с другой строкой.

```js
console.log("Mango" + 55); // "Mango55"
console.log("Mango" + true); // "Mangotrue"
```

Порядок операндов имеет значение.

```js
console.log(1 + "2"); // "12"
console.log(1 + "2" + 4); // "124"
console.log(1 + 2 + "4"); // "34"
```

<center><font color="#ff0000">В последнем примере сначала выполняется математическое сложение для первых двух чисел 1 и 2, затем число 3 превращается в строку "3" и сочетается со строкой "4".</font></center>

## Преобразование типов: строки

Преобразование типов в JavaScript — это процесс изменения значения одного типа данных в другой тип данных

### Явное преобразование типов

```js
console.log(String(5)); // "5"
console.log(String(true)); // "true"
console.log(String(false)); // "false"
console.log(String(null)); // "null"
console.log(String(undefined)); // "undefined"
```

### Неявное преобразование типов

```js
console.log("5" + 3); // "53"
console.log("5" + true); // "5true"
console.log("5" + false); // "5false"
console.log("5" + null); // "5null"
console.log("5" + undefined); // "5undefined"
```

## Шаблонные строки

Шаблонные строки - это синтаксис который поможет создавать строки и вставлять внутрь переменные.

<center><font color="#ff0000">Важно поместить такую строку в одинарные обратные кавычки `(``)`</font></center>

Название переменной мы должны обернуть в `${имя переменной}`

```js
const guestName = "Mango";
const roomNumber = 207;
const greeting = `Welcome ${guestName}, your room number is ${roomNumber}!`;
console.log(greeting); // "Welcome Mango, your room number is 207!"
```

<center><font color="#00b0f0">В основе мы используем шаблонные строки чем конкатенацию так как он более читабельный</font></center>

## Индексация строк

Строки – это наборы символов, где каждый символ имеет свой порядковый номер (индекс). Индексация элементов строки начинается с нуля. Первый символ имеет индекс 0, второй – индекс 1, третий – 2 и так далее.

![[Pasted image 20241203002604.png]]

Для доступа к определенному символу строки мы используем синтаксис квадратных скобок, где указываем индекс нужного символа: `string[index]`, то есть `строка[индекс]`.

```js
const product = "Repair droid";
console.log(product[0]); // 'R'
console.log(product[5]); // 'r'
console.log(product[11]); // 'd'
```

Получение последнего символа строки возможно по индексу `string[lastIndex]`. Чтобы найти индекс последнего символа строки, нужно из длины этой строки отнять единицу `string.length - 1`


```js
const product = "Repair droid";
const lastElementIndex = product.length - 1;
console.log(product[lastElementIndex]); // 'd'
```

Можем сократить код и записать в квадратных скобках сразу уравнение `string.length - 1`

```js
const product = "Repair droid";
console.log(product[product.length - 1]); // 'd'
```

## Неизменность строк

С помощью индекса мы не можем менять отдельные символы в строках

```js
let product = "Droid";
console.log(product); // "Droid"

// Це не має жодного ефекту ❌
product[2] = "O";
console.log(product); // "Droid"
```

```js
let product = "Droid";
console.log(product); // "Droid"

// Работает ✅
product = "DrOid";
console.log(product); // "DrOid"
```
