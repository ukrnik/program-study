<font color="#fcb97d">`ris:Question`</font>При виконанні операцій з дробовими числами можуть виникати неточності через внутрішнє представлення чисел у пам'яті комп'ютера.

```js
console.log(0.1 + 0.2 === 0.3); // false
console.log(0.1 + 0.2); // 0.30000000000000004
```

---

## Подход 1

- Умножить числа на достаточно большое число (например, 10 или 100).
- Выполнить добавление.
- Разделить результат на то же число, чтобы вернуть его к исходному масштабу.

```js
console.log(0.1 * 10 + 0.2 * 10); // 3
console.log((0.1 * 10 + 0.2 * 10) / 10); // 0.3
```

---
## Подход 2

Использовать метод числа [[Метод toFixed()| toFixed()]] для округления результата до определенного числа знаков после точки.

```js
console.log((0.1 + 0.2).toFixed(1)); // "0.3"
console.log((5).toFixed(2));  //  ”5.00”
console.log((8.762195).toFixed(4));  //  “8.7622”
```
