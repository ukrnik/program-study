Возвращает случайное число в диапазоне от 0 (включительно) до 1 (за исключением).

```js
console.log(Math.random()); // случайное число между 0 и 1, например 0.2 ... 0.9166353649342294
```

Метод Math.random() в JavaScript не принимает никаких параметров. Этот метод возвращает псевдослучайное число с плавающей запятой в диапазоне от 0 (включительно) до 1 (исключительно).

Чтобы получить случайное число в заданном диапазоне, можно использовать следующий способ:

```js
let randomInt = Math.floor(Math.random() * (10 - 5 + 1)) + 5;
console.log(randomInt); // Целое число от 5 до 10 включительно
```

1. Math.random() генерирует число от 0 до 1.
2. `(max - min)` масштабирует диапазон.
3. `+ min` сдвигает число к нужному минимуму.
4. Math.floor() округляет вниз для получения целого числа.
  
Таким образом, можно программно задавать диапазон.
