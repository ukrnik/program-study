Класс Math является встроенным JavaScript. Этот класс предоставляет набор методов выполнения математических операций и работы с числами.

| [[Метод Math.floor(num)]]           | Возвращает ближайшее целое число, которое меньше или равно  |
| ----------------------------------- | ----------------------------------------------------------- |
| [[Метод Math.ceil(num)]]            | Возвращает ближайшее целое число, которое больше, или равно |
| [[Метод Math.round(num)]]           | Округления до ближайшего целого.                            |
| [[Метод Math.max(num1, num2, ...)]] | Возвращает наибольшее число из набора                       |
| [[Метод Math.min(num1, num2, ...)]] | Возвращает наименьшее число из набора                       |
| [[Метод Math.random()]]             | Возвращает случайное число в диапазоне                      |
