# Асинхронность - setTimeout

## Определение
`setTimeout` — встроенная функция JavaScript, которая откладывает выполнение переданной функции на заданное время (в миллисекундах), **не блокируя** основной поток.

---
## Синтаксис

```js
setTimeout(callback, delay, arg1, arg2, ...)
```

- **callback** — функция, которая будет выполнена после задержки.
- **delay** — задержка в миллисекундах (1000 мс = 1 секунда).
- **...args** — дополнительные аргументы, которые будут переданы в callback.

---
## **Возвращаемое значение**

> [!warning]
> Вызов setTimeout() возвращает **идентификатор таймера**:
> - В **браузере** — это целое число (1, 2, 3…).
> - В **Node.js** — это объект Timeout, который можно передать в clearTimeout().

Этот ID используется для отмены запланированного выполнения.

```js
const timerId = setTimeout(() => {
  console.log('Это сообщение не появится');
}, 3000);

console.log(timerId); // Например: 1 в браузере

clearTimeout(timerId); // Отмена таймера
```


```js
const greet = () => {
  console.log("Hello!");
};

const timerId = setTimeout(greet, 3000);

clearTimeout(timerId); // Отмена таймера
```


---
## **Особенности**

> [!warning] 
> - Contents- Минимальная задержка в браузерах может быть не меньше 4 мс (по стандарту HTML5).
> - Время задержки **не гарантируется точно** — выполнение зависит от загруженности стека и работы Event Loop.
> - Задачи setTimeout попадают в **Macrotask Queue**.

---
## **Где использовать**

> [!info]
> - Отложенные действия (анимации, уведомления, подсказки)
> - Эмуляция сетевых задержек
> - Планирование одноразового события в будущем

---
## **Примеры**

### **Базовый пример**

```js
console.log('Начало');

setTimeout(() => {
  console.log('Выполнено через 2 секунды');
}, 2000);

console.log('Конец');
```

**Результат в консоли:**

```
Начало
Конец
Выполнено через 2 секунды
```

---
### **Передача аргументов в callback**

```js
function greet(name) {
  console.log(`Привет, ${name}!`);
}

setTimeout(greet, 1000, 'Света');
```

---
### **Отмена таймера**

```js
const timerId = setTimeout(() => {
  console.log('Это не будет выведено');
}, 3000);

clearTimeout(timerId); // Отмена таймера
```
