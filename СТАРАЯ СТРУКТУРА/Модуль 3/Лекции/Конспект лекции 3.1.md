### [[Метод .slice()]] - возвращает срезаные новый масив. slice(начало среза, конец)

```js
[7, 3, 8, 9, 2].slice(3) // новый масив [9, 2]
```

### [[Метод .concat()]] - обьеденяет несколько масивов в один

```js
[2, 4, 6].concat([1, 3], [5]) // новый масив [2, 4, 6, 1, 3, 5]
```

### [[Метод includes()]] - проверяет есть ли заданое число или строка в масиве, возвращает true если есть и false если нет

```js
[3, 5, 12, 7, 4].includes(8) // false
```

```js

console.log(courses); // ['HTML', 'CSS', 'JavaScript', 'React', 'Python']
```

Властивість .length - возвращает число обьектов в масиве

```js
const courses = ['HTML', 'CSS', 'JavaScript', 'React', 'PostgreSQL'];
const coursesLength = courses.length;
console.log(coursesLength); // 5
```

lastIndex - это метод при помощи которого можно вычеслить последний индекс в масиве. Вычесление происходят (длина масива - 1)

```js
const courses = ['HTML', 'CSS', 'JavaScript', 'React', 'PostgreSQL'];

const coursesLength = courses.length;
console.log(coursesLength); // 5

const lastCoursesIndex = coursesLength - 1;
console.log(lastCoursesIndex); // 4

console.log(courses[lastCoursesIndex]); // 'PostgreSQL'

console.log(courses[courses.length - 1]); // 'PostgreSQL' альтернатива чтобы сократить код
```

Передача за посиланням

```js
const a = 1;
const b = a; // мы копируем значений индификатора a в b
console.log(a); // 1
console.log(b += 1); // 2 так как мы скопировали значение в b то это независимая копия и над ней можно делать всё что угодне без последствий для индификатора a

---------------------------------------------------------------------------

const a = [1, 2, 3];
const b = a;
console.log(a); // [1, 2, 3]
console.log(b); // [1, 2, 3] мы ссылаемся на масив a

a[0] = 500;
console.log(a); // [500, 2, 3]
console.log(b); // [500, 2, 3] так как мы ссылаемся на индификатор a то ранее перезаписаное значение тоже будет транслироватся в индификаторе b

---------------------------------------------------------------------------

console.log(a === b); // true так как мы индификатор b ссылается на масив как и индификатор a то метод будет возвращать true (ссылки на масив одинаковые)

---------------------------------------------------------------------------

console.log([] === []); // false так как мы сравневаем не значения в масиве а ссылки на них

console,Log([1, 2, 3] == [1, 2, 31) // false так как мы сравневаем не значения в масиве а ссылки на них
```
