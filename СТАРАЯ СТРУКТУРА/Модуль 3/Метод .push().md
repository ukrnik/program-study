.push() позволяет нам добавлять элементы в конец масива.

В скобки () мы передаем те элементы которые хотим добавить в конец масива.

```js
const courses = ['HTML', 'CSS', 'JavaScript'];
courses.push('React', 'PostgreSQL', 0, 2, 55)

console.log(courses); // ['HTML', 'CSS', 'JavaScript', 'React', 'PostgreSQL', 0, 2, 55]
```
