## Описание

**Деструктуризация** – это удобный способ извлечения значений из массивов и объектов в переменные.

> [!info] **Простыми словами:**
> Вместо того чтобы обращаться к свойствам объектов или элементам массива по отдельности, можно сразу «разложить» их в переменные.

---
## Синтаксис

**Без деструктуризации** работа со свойствами объекта выглядит так:

```js
const user = { 
	name: "Jacob", 
	age: 32 
};

console.log(user.name); // Jacob
console.log(user.age); // 32
```

**С деструктуризацией** работа со свойствами объекта выглядит так:

```js
const user = { 
	name: "Jacob", 
	age: 32 
};

const { name, age } = user;
console.log(name); // Jacob
console.log(age); // 32

```

## ИТОГ:

✔ **Деструктуризация массивов** – удобно извлекать элементы массива в переменные.
✔ **Деструктуризация объектов** – быстро получаем свойства объекта в переменные.
✔ **Можно переименовывать переменные** и **задавать значения по умолчанию**.
✔ **Используется в функциях** для удобства работы с объектами.
