## HTML списки (`<ul>``<ol>``<li>`)

### Назначение

Списки используются для структурирования информации и удобного отображения коллекций данных на веб-странице.

Список — это контейнер, внутри которого могут находиться только элементы списка — теги `<li>` (_list item_).

Списки бывают двух типов:

1. **Нумерованные (упорядоченные)** — с помощью тега `<ol>`
2. **Маркёрные (неупорядоченные)** — с помощью тега `<ul>`

---

### **Нумерованный список** `<ol>`

  

Тег `<ol>` (_ordered list_) создаёт список, элементы которого нумеруются браузером автоматически.

Такой список используется, когда важен **порядок действий** или **очерёдность пунктов**.

  

**Пример:**

```html
<h1>Как заварить чай</h1>
<p>Пошаговая инструкция для начинающих.</p>

<ol>
  <li>Закипяти воду</li>
  <li>Положи чай в чашку</li>
  <li>Налей кипяток</li>
  <li>Подожди 10 минут и наслаждайся</li>
</ol>
```

Браузер сам присваивает каждому элементу `<li>` номер в порядке следования.

Если удалить пункт — нумерация обновится автоматически.

---
### **Маркёрный список** `<ul>`

  

Тег `<ul>` (_unordered list_) создаёт список, где каждый элемент отмечен маркером (точкой, кружком, квадратом и т.д.).

Используется, когда **порядок элементов не имеет значения**.

  

**Пример:**

```html
<h1>Самые жаркие курорты</h1>
<p>В этом году эксперты советуют посетить следующие локации:</p>

<ul>
  <li>Тунис</li>
  <li>Турция</li>
  <li>Греция</li>
  <li>Египет</li>
</ul>
```

Маркеры можно изменить или убрать с помощью CSS:

```html
ul {
  list-style-type: square; /* тип маркера: disc | circle | square | none */
}
```

---
### **Элементы списка** 

### `<li>`

  

Каждый пункт списка создаётся с помощью тега `<li>`.

Он должен быть **вложен** внутрь `<ul>` или `<ol>`.

  

**Пример списка ингредиентов:**

```html
<p>
  Выбор правильного сыра очень важен для этого блюда. Лучше всего подойдёт фермерский сыр.
  Также можно использовать рикотту или творог — вкус будет немного отличаться.
</p>

<ul>
  <li>2 cups farmer's cheese, homemade</li>
  <li>3 large eggs</li>
  <li>3 cups all-purpose flour, plus about 1/2 cup more for dredging</li>
  <li>3 tablespoons white sugar</li>
  <li>half teaspoon salt</li>
  <li>2–3 tablespoons extra light olive oil for each batch</li>
  <li>fresh fruit, jam, or sour cream for topping</li>
</ul>
```

---
### **Вложенные списки**

Внутри элемента `<li>` можно разместить ещё один `<ul>` или `<ol>`.
Так создаются **вложенные списки**.

  
**Пример:**

```html
<ul>
  <li>Фрукты
    <ul>
      <li>Яблоки</li>
      <li>Бананы</li>
      <li>Апельсины</li>
    </ul>
  </li>
  <li>Овощи</li>
</ul>
```

---
### **Правильное оформление кода**

- Элементы списка должны иметь **одинаковый отступ** (обычно 2 или 4 пробела).
- Отступы визуально показывают структуру документа и облегчают чтение кода.
- Теги должны быть **закрыты правильно**: `<ul>` и `<ol>` закрываются после всех `<li>`.

---
### **Применение списков в веб-разработке**

- Меню навигации сайта
- Перечни шагов, инструкций, рецептов
- Списки товаров или характеристик
- Компоненты интерфейсов (например, выпадающие списки, чек-листы)

---
### **Итог**

| **Тег** | **Назначение**                                              |
| ------- | ----------------------------------------------------------- |
| `<ul>`  | Создаёт неупорядоченный (маркёрный) список                  |
| `<ol>`  | Создаёт упорядоченный (нумерованный) список                 |
| `<li>`  | Элемент списка (должен быть дочерним для `<ul>` или `<ol>`) |
