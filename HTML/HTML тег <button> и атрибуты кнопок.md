## **HTML тег** `<button>`**и атрибуты кнопок**

## **Назначение**

> [!info]
> Тег `<button>` используется для создания интерактивных кнопок на веб-странице.
> 
> Кнопки применяются для отправки данных формы, запуска JavaScript-функций, переключения состояния элементов и других действий.

---
## **Синтаксис**

```html
<button>Текст кнопки</button>
```

> [!warning]
> Браузер по умолчанию отображает кнопку с системным стилем.
С помощью CSS её внешний вид можно изменить полностью.

---
## **Основные атрибуты тега**

| **Атрибут**    | **Назначение**                                                                    |
| -------------- | --------------------------------------------------------------------------------- |
| type           | Определяет тип кнопки                                                             |
| name           | Имя кнопки (используется при отправке формы)                                      |
| value          | Значение, которое отправляется при нажатии                                        |
| disabled       | Делает кнопку неактивной                                                          |
| autofocus      | Устанавливает автоматическое фокусирование при загрузке страницы                  |
| form           | Связывает кнопку с определённой формой (по её id)                                 |
| formaction     | Указывает адрес, на который будет отправлена форма при нажатии именно этой кнопки |
| formmethod     | Определяет метод отправки (get или post) для данной кнопки                        |
| formtarget     | Определяет, где откроется результат отправки (_self, _blank и др.)                |
| formnovalidate | Отменяет проверку данных формы перед отправкой                                    |

---
### **Атрибут** type

> [!warning]
> Атрибут type задаёт поведение кнопки. Может принимать три значения:

|**Значение**|**Описание**|
|---|---|
|button|Обычная кнопка, не выполняет действий по умолчанию|
|submit|Отправляет данные формы (значение по умолчанию, если кнопка находится в форме)|
|reset|Сбрасывает данные формы до исходных значений|

```html
<!-- Кнопка без действий -->
<button type="button">Нажми меня</button>

<!-- Кнопка для отправки формы -->
<button type="submit">Отправить</button>

<!-- Кнопка для сброса формы -->
<button type="reset">Сбросить</button>
```

---
### Атрибут name и value

> [!warning]
> Используются при отправке формы.
> 
> Если у кнопки есть атрибуты name и value, они передаются на сервер как пара ключ-значение при отправке формы.

```html
<form action="/result" method="get">
  <button type="submit" name="action" value="save">Сохранить</button>
  <button type="submit" name="action" value="delete">Удалить</button>
</form>
```

При нажатии «Сохранить» на сервер уйдёт action=save, а при «Удалить» — action=delete.

---
### **Кнопки с атрибутом** disabled

> [!warning]
> Атрибут disabled делает кнопку неактивной.
Пользователь не может на неё нажать, и она обычно отображается в сером цвете.

```html
<button type="button" disabled>Недоступно</button>
```

---
### Атрибут autofocus

> [!warning]
> Задаёт автоматическое фокусирование на кнопку при загрузке страницы.
> Может использоваться только **один раз на странице**.

```html
<button type="submit" autofocus>Отправить</button>
```

---
### Атрибут form

Позволяет связать кнопку с формой, даже если она находится **вне** тега `<form>`.

```html
<form id="feedbackForm" action="/feedback" method="post">
  <input type="text" name="name" placeholder="Ваше имя" />
</form>

<button type="submit" form="feedbackForm">Отправить отзыв</button>
```

---
### Атрибут formaction

Переопределяет адрес отправки формы (атрибут action в теге `<form>`) **только для этой кнопки**.

```html
<form action="/register" method="post">
  <button type="submit">Зарегистрироваться</button>
  <button type="submit" formaction="/login">Войти</button>
</form>
```

---
### Атрибут formtarget

Определяет, где откроется результат отправки формы.

|**Значение**|**Поведение**|
|---|---|
|_self|Открыть в текущей вкладке (по умолчанию)|
|_blank|Открыть в новой вкладке|
|_parent|Открыть в родительском фрейме|
|_top|Открыть во всём окне браузера|

```html
<form action="/result">
  <button type="submit" formtarget="_blank">Открыть в новой вкладке</button>
</form>
```

---
### Атрибут formnovalidate

> [!info]
> Отключает проверку данных формы при нажатии на кнопку.
Полезно, если нужно отправить форму, даже если в ней не заполнены обязательные поля.

```html
<form action="/register">
  <input type="email" name="email" required placeholder="Введите email" />
  <button type="submit">Отправить с проверкой</button>
  <button type="submit" formnovalidate>Отправить без проверки</button>
</form>
```


---
### **Вложенный контент**

> [!warning]
> В отличие от тега `<input type="button">`, тег `<button>` может содержать **любой HTML-контент**:
текст, изображения, иконки, спаны и даже SVG-графику.

```html
<button type="button">
  <img src="icon.png" alt="иконка" width="16" height="16" />
  Сохранить
</button>
```

---
### **Использование с JavaScript**

> [!warning]
> Кнопка часто используется для вызова функций при нажатии.
Это можно сделать через атрибут onclick или через обработчик события.

```html
<button type="button" onclick="alert('Кнопка нажата!')">Нажми меня</button>
```

**Через JavaScript:**

```html
<button id="myButton" type="button">Пуск</button>

<script>
  const button = document.getElementById('myButton');
  button.addEventListener('click', () => {
    console.log('Кнопка нажата!');
  });
</script>
```

---
### **Кнопка вне формы**

> [!warning]
> Если кнопка расположена **вне** тега `<form>`, но должна управлять формой — её можно связать с формой с помощью атрибута form.

```html
<form id="userForm" action="/submit" method="post">
  <input type="text" name="username" placeholder="Имя" />
</form>

<button type="submit" form="userForm">Отправить форму</button>
```

---
### **Стилизация кнопок**

> [!warning]
> По умолчанию внешний вид кнопок зависит от браузера и операционной системы.
Для единообразия в дизайне их обычно стилизуют с помощью CSS.

```html
button {
  background-color: #007bff;
  color: white;
  border: none;
  padding: 10px 16px;
  border-radius: 6px;
  cursor: pointer;
  font-size: 16px;
}

button:hover {
  background-color: #0056b3;
}
```

---
### **Разница между** `<button>`и `<input type="button">`

| **Характеристика**     | `<button>`                                                | `<input type="button">`       |
| ---------------------- | --------------------------------------------------------- | ----------------------------- |
| Содержимое             | Может содержать HTML-контент (текст, изображения, иконки) | Только текст в атрибуте value |
| Гибкость оформления    | Более гибкий                                              | Ограниченный                  |
| Использование в формах | Поддерживает все атрибуты формы                           | Частично                      |
| Рекомендуется          | ✅ Современный стандарт                                    | ⚠️ Устаревающий вариант       |

---
### **Итог**

| **Тег**                 | **Назначение**                                       |
| ----------------------- | ---------------------------------------------------- |
| `<button>`              | Создаёт интерактивную кнопку                         |
| `<input type="button">` | Альтернативный способ создания кнопки (ограниченный) |
