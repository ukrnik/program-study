### Задача 3

```js
const values = [10, 3, 17, 84, 12, 6];

const min = Math.min(...values); // (10, 3, 17, 84, 12, 6)

console.log(min); // 3
```

Консоль выведет 3 так как используется метод распила [[Синтаксис spread і rest]]. По итогу ...values будет набор отдельных цифр `(10, 3, 17, 84, 12, 6)` 

Если бы мы передали просто значение `values` то мы бы передали масив из чисел, а метод `Math.min()` не работает с [[Масиви]].

```js
const values = [10, 3, 17, 84, 12, 6];

const min = Math.min(values); [10, 3, 17, 84, 12, 6]

console.log(min); // NaN
```

## Лекция 

Так как объект это сборище информации то нужно как-то работать с ними. Метод [[Об’єкт]] это функция написанная внутри объекта

```js
const playlist = {
	name: 'My amazing playlist',
	rating: 5,
	tracks: ['track-1', 'track-2', 'track-3'],
	changeName() {},
	addTrack(track) {},
	updatteRating(newRating) {},
}
```

С методами объекта лучше использовать ключевое слово `this` он будет работать только внутри с этим объектом без жесткой привязки к объекту

![[Пример this.png]]

По итогу мы можем использовать `this` для облегчения роботы с вложенными функциями

```js
const playlist = {
	name: 'My amazing playlist',
	rating: 5,
	tracks: ['track-1', 'track-2', 'track-3'],
	changeName(name) {
		this.name = name;
	},
	addTrack(track) {
		this.tracks.push(track);
	},
	updatteRating(newRating) {
		this.rating = newRating;
	},
}
```

---

Бывает так что в функцию мы можем передавать разное количество данных, мы можем собрать все эти данные в [[Масиви]] через синтаксис [[Синтаксис spread і rest]] для дальнейшей роботы с ними

```js
function multiply(first, second, ...args) {
  console.log(first, second, args);
}

multiply(1, 2); // 1 2
multiply(1, 2, 3); // 1 2 [3] 
multiply(1, 2, 3, 4); // 1 2 [3, 4]
```

---

Создание нового [[Масиви]] при помощи [[Синтаксис spread і rest]].

```js
const newArray = [...copyArray];
```

после троеточие мы вставляем название масива который хотим скопировать. Это будет полностью независимая копия

Мы также можем соединять масивы при помощи синтаксиса

```js
const newArray = [...copyArray1, ...copyArray2];
```

---

Метод Math не работает с масивами, поэтому мы можем использовать [[Синтаксис spread і rest]] для передачи чисел.

```js
const temps = [18, 14, 12, 21, 17, 29, 24];

let max = Math.max(temps); // ❌ NaN

let max = Math.max(...temps); // ✅ 29

let min = Math.min(...temps); // ✅ 12
```
