### Задача 5

```js
const userA = {
	username: "Mango",
	age: 20
};
const userB = userA;
userB.age = 30;
console.log(userA.age, userB.age); // 30, 30
```

В памяти создается библиотека из ключей и свойств. userB получает ссылку на одну и туже библиотеку что и userA и поэтому когда мы перезаписываем `userB.age = 30;`то библиотека переписывается, и когда мы через userA просим данные из объекта то там уже измененное свойство

Объект создается таким способом

```js

const playlist = {
```

### Доступ к свойствам объекта

Есть два варианта как добраться до свойства в объекте, это:
1. Через точку
2. Через квадратные скобки

1 Вариант: Чтобы получить свойство из объекта то мы используем синтаксис обращения к название объекта, точка, и ключ который нам нужен

```js
const playlist = {
	Name: 'My favorite songs',
	Rating: 4.8,
	Tracks: ['track-1', 'track-2', 'track-3'],
	IsFavorite: true
	ganres: [
		{
			rock: ['Thunder'],
		},
		{
			pop: ['I believe I can fly'],
		},
	]
};

console.log(playlist.Name); // 'My favorite songs'
console.log(playlist.Rating); // 4.8
console.log(playlist.Tracks); // ['track-1', 'track-2', 'track-3']
console.log(playlist.Tracks[1]); // track-2

console.log(playlist.ganres[0].rock); // Thunder
```

2 Вариант: Это обращение к объекту через квадратные скобки

```js
const playlist = {
	Name: 'My favorite songs',
	Rating: 4.8,
	Tracks: ['track-1', 'track-2', 'track-3'],
	IsFavorite: true
	ganres: [
		{
			rock: ['Thunder'],
		},
		{
			pop: ['I believe I can fly'],
		},
	]
};

const songName = playlist['Name'];
console.log(songName); // My favorite songs
```

Второй вариант лучше использовать когда мы не знаем какое свойство мы будем получать то тогда используем этот вариант.

### Замена или обновление свойств в объекте происходит через указание объекта 1 или 2 вариантом и присвоении нового значения

```js
const playlist = {
	Name: 'My favorite songs',
	Rating: 4.8,
	Tracks: ['track-1', 'track-2', 'track-3'],
	IsFavorite: true
};

playlist.Rating = 5; // 1 вариант

playlist['Rating'] = 5; // 2 вариант

console.log(playlist); // Name: 'My favorite songs'
                       // Rating: 5
	                   // Tracks: ['track-1', 'track-2', 'track-3']
	                   // IsFavorite: true
```

### Короткие свойства - это сокращенный синтаксис присвоения значения

```js
const username = 'Mango';
const email = 'mango@mail.com';

const credentials = {
	username,
	email,
};

console.log(credentials); // {username: 'Mango', email: 'mango@mail.com'}
```

Это нужно для того чтобы не записывать два раза одно и тоже

```js
const username = 'Mango';
const email = 'mango@mail.com';

const credentials = {
	username: username,
	email: email,
};

console.log(credentials); // {username: 'Mango', email: 'mango@mail.com'}
```

Если мы хотим подставить значение как ключ то мы используем синтаксис квадратных скобок

```js
const inputName = 'color';

const colorPickerData = {
	[inputName]: 'tomato',
};

console.log(colorPickerData); // {color: 'tomato'}
```

### Задачка из лекции

- Створи функцію яка буде приймати 3 аргументи та формувати обʼєкт покупки
- ﻿﻿1 Назва продукту
- ﻿﻿2 Кількість одиниць
- ﻿﻿3 Ціна за 1 одиницю товару
- ﻿﻿Функція має повертати сформований обʼєкт з ключами name, price, quantity, totalPrice.

```js
function createBasket(name, price, quantity) {
	console.log(name, price, quantity); // drones 50 100000
	const basket = {
		productName: name,
		productPrice: price,
		productQuantity: quantity,
		totalPrice: price * quantity,
	};
	console.log(basket); // {productName: 'drones', productPrice: 50,                                    productQuantity: 100000, totalPrice: 5000000}
};

createBasket('drones', 50, 100_000);
```

Также исходя из коротких свойств мы можем упростить код

```js
function createBasket(name, price, quantity) {
	console.log(name, price, quantity); // drones 50 100000
	const basket = {
		name,
		price,
		quantity,
		totalPrice: price * quantity,
	};
	console.log(basket); // {name: 'drones', price: 50, quantity: 100000}
};

createBasket('drones', 50, 100_000);
```

Конечное решение этой задачи такое:

```js
function createBasket(name, price, quantity) {
	return {
		name,
		price,
		quantity,
		totalPrice: price * quantity,
	};
};

createBasket('drones', 50, 100_000); // {name: 'drones', price: 50, quantity: 100000, totalPrice: 5000000,}
```

### Перебор объекта с помощью цикла [[Цикл for...in]]


```js
const feedback = {
	good: 5, 
	neutral: 10, 
	bad: 3,
}

for (const key in feedback){
	console.log(key) // good // neutral // bad
}
```

Мы используем цикл for...in потому-что в цикле for мы подставляем число счетчика в индекс для перебора масива, а в объекте у нас не индексы а ключи и поэтому нужно использовать цикл for...in (мы как-бы проходимся по каждому ключу)

### Чтобы получить значения ключа в объекте мы используем синтаксис обращения через квадратные скобки

```js
const feedback = {

	bad: 3,

for (const key in feedback){
```

### [[Метод Object.keys()]] создаёт масив из ключей объекта

```js
const feedback = {
	good: 5, 
	neutral: 10, 
	bad: 3,
}

const keys = Object.keys(feedback);
console.log(keys); // ['good', 'neutral', 'bad']
```

и теперь мы можем использовать [[Цикл for...of]] так как это масив из ключей

```js
const feedback = {
	good: 5, 
	neutral: 10, 
	bad: 3,
}

const keys = Object.keys(feedback);

for (const key of keys) {
	console.log(key);            // good // neutral // bad
	console.log(feedback[key]);  // 5 // 10 // 3
}
```

---
### [[Метод Object.values()]] создаёт масив из значений которые в объекте

```js
const feedback = {
	good: 5, 
	neutral: 10, 
	bad: 3,
}

const keys = Object.values(feedback);
console.log(keys); // [5, 10, 3]
```

---
### [[Метод Object.entries()]] создаёт масив масивов где первое будет ключ а дальше значение этого ключа

```js
const feedback = {
	good: 5, 
	neutral: 10, 
	bad: 3,
}

const keys = Object.entries(feedback);
console.log(keys); // [['good', 5], ['neutral', 10], ['bad', 3]]
console.log(keys[1]); // ['neutral', 10]
console.log(keys[1][0]); // neutral
```
