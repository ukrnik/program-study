## Описание

**switch** — это **конструкция** для выполнения разных **блоков кода** в зависимости от значения выражения. Это альтернатива множественным **if...else if...else**.

---
## Синтаксис

```js
switch (выражение) {
  case значение1:
    // Код, если выражение === значение1
    break;
  case значение2:
    // Код, если выражение === значение2
    break;
  default:
    // Код, если ни одно из значений не совпало
}
```

- **выражение** — значение, которое сравнивается с каждым case.
- **case** — возможное значение для сравнения.
- **break** — завершает выполнение switch. Если его не написать, выполнение продолжится до следующего case.
- **default** — выполняется, если ни одно из значений не совпало.
---
## Пример №1

```js
const score = 85;

switch (true) {
  case score >= 90:
    console.log("Отлично");
    break;
  case score >= 75:
    console.log("Хорошо");
    break;
  case score >= 50:
    console.log("Удовлетворительно");
    break;
  default:
    console.log("Неудовлетворительно");
}
```

- **true** — это выражение, которое будет сравниваться с каждым case.
- **Если score >= 75** — выполнится "**Хорошо**" и остановится из-за break.
---
## Пример №2

```js
const day = "вторник";

switch (day) {
  case "понедельник":
    console.log("Начало недели");
    break;
  case "вторник":
  case "среда":
  case "четверг":
    console.log("Рабочий день");
    break;
  case "пятница":
    console.log("Скоро выходные");
    break;
  case "суббота":
  case "воскресенье":
    console.log("Выходной");
    break;
  default:
    console.log("Некорректный день");
}
```

- Здесь "**вторник**", "**среда**" и "**четверг**" объединены в один блок.
- Если **day** = "**вторник**", то выведется "**Рабочий день**".
