Строгий режим (англ. strict mode) в JavaScript — это специальный режим выполнения кода, позволяющий переводить скрипт в режим полного соответствия современному стандарту. Это предотвращает определенные ошибки, например использование опасных и устаревших конструкций, что делает код более надежным и позволяет легче его поддерживать.

Чтобы включить строгий режим, необходимо добавить директиву `'use strict'` до начала скрипта. Например:

```js
'use strict';

// Код в строгом режиме
```
