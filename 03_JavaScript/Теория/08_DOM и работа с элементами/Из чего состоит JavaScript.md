JavaScript при роботе с браузером имеет дополнительный функционал для роботы с вкладкой. Этот функционал складывается из трёх модулей (ECMAScript, DOM, BOM.)

![[JsModules.jpg]]

ECMAScript или (ES) - это то что являеться самим языком. Синтаксис, функции, объэкты для роботы с данными и тд.

Document Object Model или ([[DOM]]) - это независимый функционал что соединяет роботу станицы html и JS. С помощь DOM мы можем искать, удалять, создавать елементы html и менять значения под нужды пользователя. Структура DOM это каждый елемент html (абзац, ссылка, заголовок, span и тд.) и все они могут быть измененнми с помощь JavaScript. 

Browser Object Model или (BOM) - это тоже независимый функционал от языка для роботы с браузером. С помошью BOM мы можем получить доступ к ряду функций браузера а именно читать URL, розмеры окна, читать историю навигации, местом нахождения окна и тд.
