Возвращает значение числа после округления до ближайшего целого. Округление происходит по математическим правилам, если десятичная часть числа меньше 0.5, то округление будет в меньшую сторону, если 0.5 и больше – то в большую.

```js
console.log(Math.round(1.3)); // 1
console.log(Math.round(1.7)); // 2
```
