## Арифметические операторы

| Оператор | Пример     | Результат       | Описание                         |
|----------|-----------|----------------|-----------------------------------|
| `+`      | `8 + 5`   | `13`           | Сложение                         |
| `-`      | `8 - 5`   | `3`            | Вычитание                        |
| `*`      | `8 * 5`   | `40`           | Умножение                        |
| `/`      | `8 / 5`   | `1.6`          | Деление                          |
| `%`      | `8 % 5`   | `3`            | Остаток от деления               |
| `**`     | `8 ** 5`  | `32768`        | Возведение в степень             |

---

Оператор `(+)` используется для сложения двух чисел.

```js
const x = 8;
const y = 5;
console.log(x + y); // 13
```

---

Оператор `(-)` используется для вычитания одного числа из другого.

```js
const x = 8;
const y = 5;
console.log(x - y); // 3

```

---

Оператор `(*)` используется для умножения двух чисел.

```js
const x = 8;
const y = 5;
console.log(x * y); // 40
```

---

Оператор (/) используется для деления одного числа на другое.

```js
const x = 8;
const y = 5;
console.log(x / y); // 1.6
```

---

Оператор (%) возвращает остаток от деления одного числа на другое.

```js
const x = 8;
const y = 5;
console.log(x % y); // 3
```

---

Оператор `(**)` используется для возведения числа в степень.

```js
const x = 8;
const y = 5;
console.log(x ** y); // 32768
```

---
## Комбинированные операторы

| Оператор | Пример     | Эквивалент     | Описание                         |
|----------|-----------|---------------|-----------------------------------|
| `+=`     | `x += y`  | `x = x + y`   | Прибавляет значение              |
| `-=`     | `x -= y`  | `x = x - y`   | Вычитает значение                |
| `*=`     | `x *= y`  | `x = x * y`   | Умножает значение                |
| `/=`     | `x /= y`  | `x = x / y`   | Делит значение                   |
| `%=`     | `x %= y`  | `x = x % y`   | Остаток от деления                |
| `**=`    | `x **= y` | `x = x ** y`  | Возводит в степень               |
