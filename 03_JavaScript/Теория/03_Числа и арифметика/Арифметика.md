## <font color="#1AC8ED">Операторы сравнения</font> - возвращают булевое значение (`true` или `false`)

- Оператор > (больше) – возвращает true, если левый операнд больше правого. В противном случае возвращает false.
- Оператор < (меньше) – возвращает true, если левый операнд меньше правого. В противном случае возвращает false.
- Оператор >= (больше или равен) — возвращает true, если левый операнд больше или равен правому. Если наоборот возвращает false .
- Оператор <= (меньше или равно) — возвращает true, если левый операнд меньше или равен правому. Если наоборот возвращает false .

```js
const a = 2;
const b = 5;

console.log(a > b); // false
console.log(b > a); // true
console.log(a >= b); // false
console.log(b >= a); // true

console.log(a < b); // true
console.log(b < a); // false
console.log(a <= b); // true
console.log(b <= a); // false
```

## <font color="#1AC8ED">Операторы равенства</font>

### <font color="#1AC8ED">Оператор не строгого равенства:</font>

- Оператор == (равно) — сравнивает два значения на равенство и возвращает true, если значения операндов равны. Если наоборот, возвращает false
- Оператор != (не равен) – сравнивает два значения на неравенство и возвращает true, если значения операндов не равны. Если наоборот, возвращает false

```js
console.log(5 == 5); // true
console.log(5 == 3); // false
console.log(5 != 3); // true
console.log(5 != 5); // false
```

<center><font color="#fcb97d">Операторы строгого равенства могут приводить значение к разным типам, в зависимости от контекста сравнения. Это может привести к неожиданным результатам, поэтому многие разработчики пытаются избегать их использования.</font></center>

```js
// Погано, бо виконується неявне приведення рядків і булевого значення до числа
console.log(5 == "5"); // true
console.log(5 != "5"); // false
console.log(1 == true); // true
console.log(1 != true); // false
```

### <font color="#1AC8ED">Оператор не строгого равенства:</font>

- Оператор === (строгое равенство) – сравнивает два значения на равенство, возвращает true, если значения операндов равны и имеют одинаковый тип данных. В противном случае, возвращает false
- Оператор !== (строгое неравенство) — сравнивает два значения на неравенство, возвращает true, если значения операндов не равны или имеют разные типы данных. В противном случае, возвращает false

```js
// Добре, приведення типів не виконується
console.log(5 === 5); // true
console.log(5 === "5"); // false
console.log(5 !== "5"); // true
console.log(5 !== 5); // false
console.log(1 === true); // false
console.log(1 !== true); // true
```

В отличие от строгих операторов, выражения `5 === "5"` и `1 === true` вернут false, потому что операнды имеют разные типы.
